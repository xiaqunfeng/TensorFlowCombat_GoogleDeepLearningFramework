## 所做的工作

- 基于当前最新版本的 tensorflow 对代码中被遗弃及更改的特性进行修正；
- 所有代码基于python3；
- 对代码增加了了必要的注释；
- 添加了自己的学习笔记。

```
>>> import tensorflow as tf
>>> tf.__version__

```

python版本：

```
▶ python3 --version

```
